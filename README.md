# Bank Reconciliation Pipeline

Data engineering for reconciling bank and accounting files.

## Description

This project implements a data pipeline that loads, processes, and compares bank and accounting data to identify discrepancies. It generates a standardized output file called `conciliacion` with the reconciliation results.

## Features

- Load bank and accounting files
- Data cleaning and transformation
- Automated reconciliation logic
- Output generation (`conciliacion`)
- Cross-platform support (Windows/Linux)

## Requirements

- Python 3.x
- pandas

Install dependencies:

```bash
pip install pandas
```
